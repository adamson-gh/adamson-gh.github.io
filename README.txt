# PDF2Text Local — Offline PDF → Text Converter

A tiny Windows tool that extracts text from PDF files **locally on your computer**  
and saves it as a `.txt` file next to the PDF.

No uploads. No internet. No tracking.

---

## ✔ What it does

- Extracts selectable text from PDFs
- Saves a `.txt` file beside the original PDF
- Runs entirely offline
- No installation required

Example:

report.pdf → report.txt

---

## ✔ How to use

1. Double-click `pdf2text-local.exe`
2. Click **Select PDF…**
3. Choose a PDF file
4. Click **Convert to .txt**

The text file will appear in the same folder as the PDF.

---

## ✔ Requirements

- Windows 10 or 11
- No Python or setup needed

---

## ✔ Privacy

All processing happens locally on your machine.  
Your files are never uploaded or shared.

---

## ⚠ Limitations (v1)

This tool extracts **embedded/selectable text** only.

Some PDFs are scans (images of pages).  
These contain no embedded text and will produce no output.

Example of scanned PDF:
- Printed document scanned to PDF
- Faxed documents
- Image-only PDFs

OCR support may be added in a future version.

---

## ✔ Typical uses

- Extract text from reports
- Copy text from PDFs
- Archive documents as text
- Prepare content for editing
- Searchable text backups

---

## ✔ Troubleshooting

**Nothing happens when converting**
→ The PDF likely contains no selectable text.

**Windows shows a security warning**
→ Click **More info → Run anyway**  
(This occurs because the tool is unsigned.)
