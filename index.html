PDF2Text Local

Offline PDF → text converter for Windows. Extracts embedded/selectable text and saves a .txt next to the PDF. No uploads. No cloud.

Download (zip) · Readme

Note: Scanned/image-only PDFs require OCR (not included in v1).

Photo Meta Extractor

Offline photo metadata (EXIF) → CSV for Windows. Extract camera, date taken, dimensions, GPS, and file info from images locally. No uploads. No cloud.

Download (zip) · Readme

Note: Works best for JPG/JPEG images with EXIF metadata. Scanned or edited images may contain little or no EXIF.
